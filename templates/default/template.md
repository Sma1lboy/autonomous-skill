# default template

Generic worker guidance with no toolchain-specific slash commands.

## Allow

- New idea? -> "Sketch the smallest version that proves the idea."
- Need implementation? -> "Build this. Reference the design doc if one exists."
- Feels fragile? -> "Audit this module for fragility and propose fixes."
- Bug? -> "Investigate root cause before patching."

## Block

- Never invoke shipping or deployment commands.
